# crm-sdk

Embeddable CRM Chat Widget SDK — TypeScript-first, framework-agnostic.

## Installation

```bash
npm install @projeto-crm-2026/crm-sdk
```

## Quick Start

```ts
import { init } from '@projeto-crm-2026/crm-sdk';

init({ workspaceId: 'YOUR_WORKSPACE_ID' });
```

## Development

```bash
# Install dependencies
npm install

# Build (outputs to dist/)
npm run build

# Watch mode
npm run dev

# Type-check without emitting files
npm run typecheck
```
